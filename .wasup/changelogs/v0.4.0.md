## [v0.4.0] - 2026-05-08

### Changed

#### Zig 0.16.0 API Migration — Full Codebase Upgrade (M1–M6)

This release migrates the entire **hoz** codebase from **Zig 0.15** to **Zig 0.16.0**, resolving **60+ compilation errors** across **12 categories** of breaking API changes.

##### M1: I/O System Architecture Redesign (21+ errors resolved)

- **Replaced 91 instances** of deprecated `Io.Threaded.new(.{})` with `Io.Threaded.global_single_threaded.io()` across 23+ files:
  - [`src/object/io.zig`](src/object/io.zig) — core I/O layer
  - [`src/cli/branch.zig`](src/cli/branch.zig), [`src/cli/commit.zig`](src/cli/commit.zig), [`src/cli/init.zig`](src/cli/init.zig) — CLI commands
  - All branch operations: [`create`](src/branch/create.zig), [`delete`](src/branch/delete.zig), [`list`](src/branch/list.zig), [`rename`](src/branch/rename.zig), [`upstream`](src/branch/upstream.zig)
- **Removed `.interface` field access** from `Io.Writer` — rewrote writer dispatch in [`src/cli/output.zig`](src/cli/output.zig) (8 call sites)
- **Replaced `readAll()` with buffered I/O** pattern using `w.buffer[0..w.end]`
- **Fixed `BranchAction` enum overflow** — expanded tag type from `u2` to `u3` to accommodate value range

##### M2: Standard Library Filesystem Reorganization (12 errors resolved)

- **Updated `std.fs.File` namespace access patterns** across:
  - [`src/index/index.zig`](src/index/index.zig) — 10 file operation sites updated
  - [`src/index/index_entry.zig`](src/index/index_entry.zig) — 5 file stat sites updated
- **Migrated directory creation API**: `makePath()` → new Zig 0.16 pattern in [`src/object/io.zig`](src/object/io.zig)
- **Updated file permission handling**: `stat.mode` → `stat.permissions` with `@enumFromInt()` conversion

##### M3: Const-Correctness Hardening (14 errors resolved)

- **Fixed `DebugAllocator` initialization** pattern in [`src/tree/builder.zig`](src/tree/builder.zig):
  ```zig
  // Before (Zig 0.15): gpa.allocator()
  // After  (Zig 0.16): gpa.allocator()
  ```
- **Resolved `*const T` vs `*T` mismatches** in [`src/checkout/switch.zig`](src/checkout/switch.zig):
  - Allocator comparison now uses proper mutable references
  - Fixed null comparison semantics for `*anyopaque` pointers
- **Updated RefStore test initialization** across 4 branch modules to use `var` instead of `const`

##### M4: Cryptographic Module Path Update (1 error resolved)

- **Migrated SHA-1 hashing** from deprecated `std.crypto.Sha1.hash` to local [`sha1.sha1()`](src/sha1.zig) implementation in [`src/index/index.zig`](src/index/index.zig)
- Ensures compatibility with Zig 0.16's restructured crypto module layout

##### M5: Object ID (OID) Type System Overhaul (6 errors resolved)

- **Renamed OID hex conversion method**: `.hex()` → [`toHex()`](src/object/oid.zig) in [`src/ref/ref.zig`](src/ref/ref.zig)
- **Added missing `.io` field** to `RefStore` struct initialization in [`src/ref/store.zig`](src/ref/store.zig) (4 test sites)
- **Fixed array-to-OID coercion**: `[20]u8` literals replaced with [`OID.zero()`](src/object/oid.zig) in tree builder
- **Corrected short OID parsing placement**: short hex strings now correctly populate beginning of byte array instead of end

##### M6: Struct Field & Type Mismatch Cleanup (3+ errors resolved)

- **Removed deprecated `.dev`, `.uid`, `.gid` fields** from `Io.File.Stat` usage across index subsystem
- **Updated timestamp field access**: `stat.mtime.seconds` → `@divTrunc(stat.mtime.nanoseconds, 1_000_000_000)`
- **Fixed optional string unwrapping**: added `.?` operator for `upstream_name` in [`src/branch/upstream.zig`](src/branch/upstream.zig)
- **Replaced unsafe pointer zeroing**: `std.mem.zeroes(RefStore)` → explicit undefined initialization in [`src/commit/head.zig`](src/commit/head.zig), [`src/commit/amend.zig`](src/commit/amend.zig)
