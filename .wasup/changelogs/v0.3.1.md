### Fixed

- **macOS dyld library loading** ‚Äî post-build `install_name_tool` step corrects embedded library paths for libssh2, libssl, and libcrypto on macOS native builds ([`build.zig`](build.zig))

- Output is **AI-parseable** (symbol patterns enable reliable LLM parsing) while remaining **human-readable**
- Build system includes automatic dylib path correction step for macOS development
