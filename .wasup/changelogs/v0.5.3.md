## [v0.5.3] - 2026-05-11

### Added

- **Branch decorations in `log` output** — `log` now displays branch pointers (`(HEAD -> main)`, `(feature)`) next to commit hashes across all output formats (`short`, `medium`, `full`, `oneline`). Introduced `buildDecorations()` to resolve refs and annotate commits with their associated branch names (`c230d0c`)
- **GitHub Actions release workflow** — Added `.github/workflows/release.yml` with cross-platform build matrix (Linux, macOS, Windows) for automated release artifact generation (`583e3d8`)
- **Deflate roundtrip test coverage** — Added 3 new tests to [`deflate.zig`](../src/compress/deflate.zig): fixed "AAAA" roundtrip, git blob roundtrip (`blob 10\0hello hoz\n`), and repeated pattern roundtrip (`"ABCDEFGH" ** 100`). Also added `zlibDecompress` helper using `std.compress.flate.Decompress` for test validation (`583e3d8`)

### Fixed

- **`log` command showing tree OID instead of commit OID** — All print methods (`printShort`, `printMedium`, `printFull`, `printOneline`) now receive and display the actual commit OID rather than the tree OID, matching standard Git behavior (`c230d0c`)
- **Deflate fixed Huffman encoding bit order** — `writeLiteral`, `writeLengthCode`, and `writeDistanceCode` now correctly reverse bits before writing (LSB-first), fixing corrupt compressed output. Added `reverseBits()` and `writeHuffmanCode()` helpers (`583e3d8`)
- **Deflate fixed block header** — BFINAL/BTYPE header now writes `1` bit + `2` bits (total 3 bits) instead of incorrectly writing two 1-bit values (`583e3d8`)
- **Deflate end-of-block code** — EOB (`256`) now encoded as `0` with 7 bits via `writeHuffmanCode` instead of raw `256` with 7 bits (`583e3d8`)
- **Distance code lookup logic** — Replaced fragile exact-match loop with range-based search using `capped_dist`, eliminating unreachable break paths and fixing distance encoding for values near boundaries (`583e3d8`)
- **Memory leaks in `rev_list`** — Added `defer commit.deinit(allocator)` after `Commit.parse` calls in `processQueue` and `printCommit` (2 sites) (`583e3d8`)
- **Indentation error in submodule diff** — Fixed misaligned `if` block inside `while` loop in `getSubmoduleStatus` that caused compile failure (`583e3d8`)
- **Code formatting across 6 files** — Fixed Zig fmt issues: `else if` line breaks in [`large_file.zig`](../src/diff/large_file.zig), slice spacing in [`shortlog.zig`](../src/cli/shortlog.zig), and redundant blank lines in [`branch.zig`](../src/cli/branch.zig) and [`checkout.zig`](../src/cli/checkout.zig) (`583e3d8`)
