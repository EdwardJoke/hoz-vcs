## [v0.3.4] - 2026-05-07

### Changed

#### Command Consolidation — `branch.zig` + `dispatcher.zig` (M1 + M2)

- **Merged `checkout` into `branch`** as subcommand `branch out` (or alias `branch checkout`):
  - New [`BranchAction.checkout`](src/cli/branch.zig) enum variant
  - Full checkout functionality preserved: branch switching (`branch out <target>`), branch creation with switch (`branch out -b <name>`), detached HEAD mode (`--detach`), force switch (`-f`), track mode (`--track`)
  - Uses existing [`CheckoutEngine`](src/checkout/checkout.zig) and [`CheckoutOptions`](src/cli/branch.zig) internally

- **Merged `switch` into `branch`** as subcommand `branch switch`:
  - New [`BranchAction.switch_branch`](src/cli/branch.zig) enum variant
  - Full switch functionality preserved: create+switch (`branch switch -c <name>`), force-create (`-C`), detached (`--detach`), force (`-f`), track (`--track`)
  - Uses existing [`BranchSwitcher`](src/checkout/switch.zig) and [`SwitchOptions`](src/checkout/switch.zig) internally

- **Dispatcher cleanup** ([`dispatcher.zig`](src/cli/dispatcher.zig)) (M3):
  - Removed `Checkout` import from `cli/checkout.zig`
  - Removed `Switch` import from `cli/switch.zig`
  - Deleted `runCheckout()` method (22 lines)
  - Deleted `runSwitch()` method (4 lines)
  - Removed 'checkout' and 'switch' from dispatch chain

#### Backward Compatibility — Deprecated Aliases (S1)

- **`checkout` command** preserved as deprecated alias → redirects to `branch out`
- **`switch` command** preserved as deprecated alias → redirects to `branch switch`
- Both print deprecation warning: `warning: '<cmd>' is deprecated, use 'branch <sub>' instead`
- Existing scripts and muscle memory continue working during transition period

#### Help Text Updates — `main.zig` (S2)

- `branch` description updated to: *"List, create, delete branches; switch (out/checkout/switch)"*
- `checkout` description marked as: *"[deprecated] Use 'branch out' instead"*
- `switch` description marked as: *"[deprecated] Use 'branch switch' instead"*
- Version string bumped to `hoz version 0.3.4`

### Fixed
