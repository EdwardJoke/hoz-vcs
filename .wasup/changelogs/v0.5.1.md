## [v0.5.1] - 2026-05-09

### Added
- **Shallow clone** via `hoz clone --depth <N>` — fetch only N commits of history instead of full history; writes `.git/shallow` boundary markers and validates depth parameters ([`shallow.zig`](../src/clone/shallow.zig), `9b83676`)
- **Partial clone** via `hoz clone --filter=<spec>` — supports three modes:
  - `--filter=blob:none` (blobless): download commits + trees only, fetch blobs on demand
  - `--filter=tree:0` (treeless): download commits only, minimal possible clone
  - `--filter=blob:limit=<size>` (blob limit): exclude blobs larger than threshold
  Includes [`FilterSpec`](../src/clone/partial.zig) parser, [`ObjectPromisor`](../src/clone/partial.zig) for on-demand remote fetching, and [`LazyBlobLoader`](../src/clone/partial.zig) with content-addressed cache (`ca80357`)
- **Network resilience module** ([`resilience.zig`](../src/network/resilience.zig)) with exponential backoff retry, `AuthError`/`NetworkError` classification, and `ResumableDownloader` supporting partial transfer resume (`f2e4c5a`)
- **Cross-platform abstraction layer** ([`platform.zig`](../src/workdir/platform.zig)) providing Windows-compatible path handling (backslash normalization, `joinPath()`), permission model abstraction (Unix chmod → Windows ACL), and symlink creation with junction point fallback (`35d8be7`)
- **Memory safety test suite** ([`memory_test.zig`](../src/final/memory_test.zig)) covering error-path cleanup verification and streaming size-limit enforcement (`8f35ea8`)

### Changed
- **Streaming object reader** in [`reader.zig`](../src/object/reader.zig) — replaced full in-memory loading with 64KB chunked I/O for large blob/pack files, preventing OOM on repositories with large objects (`8f35ea8`)
- **Pack receive** in [`pack_recv.zig`](../src/network/pack_recv.zig) — added 2GB size limit guard and proper error-path cleanup for oversized packfile downloads (`8f35ea8`)
- **Connection pool** in [`pool.zig`](../src/network/pool.zig) — rewritten with atomic state machine (Idle → InUse → Expired transitions via CAS), lock-free release(), double-check locking in pruneExpired(), and atomic statistics counters (`1643e2e`)
- **Clone options** in [`options.zig`](../src/clone/options.zig) — integrated `PartialCloneFilter` type into `CloneOptions` struct (`ca80357`)
- **Working directory cloner** in [`working_dir.zig`](../src/clone/working_dir.zig) — wired up depth validation, shallow file writing on `--depth > 0`, and boundary OID collection from remote refs (`9b83676`)
- **Transport layer** in [`transport.zig`](../src/network/transport.zig) — added resilience hook entry point for retry/resume integration (`f2e4c5a`)

### Fixed
- **Memory leaks** in [`object.zig`](../src/object/object.zig) — added `errdefer` guards throughout `Object.parse()` to ensure allocated memory is freed on all error return paths (`8f35ea8`)
- **Race conditions** in network connection pool ([`pool.zig`](../src/network/pool.zig)) — eliminated data races on shared connection state by replacing mutex-only synchronization with atomic compare-and-swap operations for state transitions (`1643e2e`)
- **Windows compilation error** in platform permission checks ([`platform.zig`](../src/workdir/platform.zig)) — resolved unused-variable discard that caused build failure on strict lint targets (`dd25d0a`)
- **Zig 0.16 ArrayList API compatibility** across 4 files — migrated all `ArrayList.init(alloc)` → `.empty`, added required allocator argument to every `append()`/`appendSlice()`/`deinit()` call in [`shallow.zig`](../src/clone/shallow.zig), [`partial.zig`](../src/clone/partial.zig), [`platform.zig`](../src/workdir/platform.zig), [`working_dir.zig`](../src/clone/working_dir.zig) (`e914782`)

### New CLI Usage

```bash
# Shallow clone — only latest commit
hoz clone --depth 1 https://github.com/user/repo

# Blobless clone — no binary files initially
hoz clone --filter=blob:none https://github.com/user/repo

# Treeless clone — commits only (smallest possible)
hoz clone --filter=tree:0 https://github.com/user/large-repo

# Exclude blobs larger than 1 MB
hoz clone --filter=blob:limit=1mb https://github.com/user/media-repo
```
