# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [v0.5.4] - 2026-05-12

### Fixed

- **install.sh** — Correct malformed command substitution that caused installation script failure on every fresh install (`5b9a3c7`)
- **openDirAbsolute relative paths** — Replace hardcoded `".git"` relative paths with absolute path construction in push and pull commands, preventing failures when runtime working directory differs (`099d582`)
- **Commit.parse memory leaks** — Add missing `defer commit.deinit()` calls in amend, rebase planner, branch ahead/behind counting, and repo filter — all callers of `Commit.parse()` now properly free allocated identity fields (`246dec5`)
- **Dead code in compress modules** — Remove unused `DeflateError` error set from deflate.zig; remove unused `crc32WithInitial`, `crc32Packfile`, and `Crc32Calculator` from crc32.zig (77 lines removed) (`fc1c12d`)
- **formatDate timezone** — Date output in log short/medium formats now uses the commit's actual timezone offset instead of always showing `+0000`, matching git log behavior (`e3d484d`)
- **Stale errdefer in parseFromParts** — Remove unreachable `errdefer allocator.free(owned_data)` after final allocation in `parseFromParts()` that risked double-free since earlier buffers were already freed on success path (`210a9de`)

### Removed
- Unused compression error set and CRC32 streaming API with zero production callers
