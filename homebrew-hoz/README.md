# Edward/homebrew-tap

Homebrew tap for **hoz**, a Git-compatible version control system written in Zig.

## Install

```bash
brew tap Edward/tap
brew install hoz
```

## Update

```bash
brew update
brew upgrade hoz
```

## Verify

```bash
brew info hoz
hoz version
```

## Uninstall

```bash
brew uninstall hoz
brew untap Edward/tap
```

## Development

SHA256 checksums are automatically verified on install. Version numbers are pulled from GitHub releases.

For more details, see the [hoz repository](https://github.com/Edward/hoz).
